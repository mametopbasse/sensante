# SenSante

Assistant de pré-diagnostic médical pour le Sénégal.

## Description

SenSante utilise le Machine Learning et un modèle IA
pour aider au pré-diagnostic des maladies courantes
comme :

- Paludisme
- Grippe
- Typhoïde

## Technologies

- FastAPI
- Machine Learning
- TailwindCSS
- Docker
- Groq / Llama 3
